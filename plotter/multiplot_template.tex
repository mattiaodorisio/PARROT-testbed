\subsection{Comparisons among indexes}
All the plots here below compare the performance of different indexes on various datasets by varying the initial dataset size. Please note, that the dataset size can change due to updates. 
All the workloads are run as a batch of $k$ operations. Each batch is repeated multiple times to obtain median of the execution time.
The workloads are:
\begin{itemize}
  \item In-distribution Lookup: searching for $k$ keys already present within the dataset.
  \item Out-of-distribution Lookup: searching for $k$ keys that do not belong to the same dataset, but whose distribution follows the same of the dataset.
  \item In-distribution Insertion: inserting $k$ keys drawn from the same distribution of the dataset.
\end{itemize}
